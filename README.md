# 🎓 Digital Transfer Certificate Management System

A web-based **Digital Transfer Certificate Management System** that automates the generation and management of student Transfer Certificates for educational institutions. This system includes **Admin Login, Student Registration, Transfer Certificate Generation**, and **MySQL database integration** using XAMPP.

---

## 📌 Features

* 🔐 Secure Admin Login Authentication
* 📝 Student Registration Module
* 🏠 Interactive Home Page
* 🧾 Automated Transfer Certificate Generation
* 💾 MySQL Database Integration
* 📱 Responsive User Interface

## ⚙️ Setup Instructions

Follow these steps to run the project on your local machine.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Digital-Transfer-Certificate-Management-System.git
```

### 2️⃣ Start XAMPP Server

* Open the XAMPP Control Panel.
* Start **Apache** and **MySQL**.

### 3️⃣ Import the Database

* Open **phpMyAdmin**.
* Create a new database.
* Import the provided `.sql` file.

### 4️⃣ Run the Project

* Copy the project folder into the **htdocs** directory.

```
C:\xampp\htdocs\
```

* Open your browser and visit:

```
http://localhost/Digital-Transfer-Certificate-Management-System/
```
